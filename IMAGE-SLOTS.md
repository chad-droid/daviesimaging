# Image Upload Reference — daviesimaging.com

Upload via the admin panel at `/admin/assets`. When assigning an image, enter the **Slot ID** exactly as shown.

Hero sections on all pages are **text-only** — they use an animated dark background, not a photo. No hero background images needed anywhere.
